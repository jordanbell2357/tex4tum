SI Units [edit]

Definition

The System of International Units is a dimension system that is based on 7 base units.

Dimension
Name		 Dimension
Symbol		 Unit
Name		 Unit
Symbol
Length L Meter \text{m}\,
Mass M Kilogram \text{kg}\,
Time T Second \text{s}\,
Current I Ampere \text{A}\,
Temperature Θ Kelvin \text{s}\,
Amount of substance N Mole \text{mol}\,
Luminous intensity J Candela \text{cd}\,

Derived Quantities

Quantity Definition Unit SI-Notation
Frequency f = \frac{c}{\lambda} Hertz \text{Hz}\, = /\text{s}\,
Force \boldsymbol F := m \cdot \boldsymbol a Newton \text{N}\, = \text{kg}\,\text{m}\,/\text{s}\,^{2}
Pressure p := \frac{\boldsymbol F_\perp}{A} Pascal \text{Pa}\, = \text{N}\,/\text{m}\,^{2} = \text{kg}\,/\text{m}\,\text{s}\,^{2}
{}^{\textstyle \text{Work,}}_{\textstyle \text{Energy}} W := \int \boldsymbol F \,\text{d}\boldsymbol s Joule \text{J}\, = \text{N}\,\text{m}\, = \text{kg}\,\text{m}\,^{2}/\text{s}\,^{2}
Power P := \frac{\mathrm dW}{\mathrm dt} Watt \text{W}\, = \text{J}\,/\text{s}\, = \text{kg}\,\text{m}\,^{2}/\text{s}\,^{3}
Voltage U := \frac{W}{Q} Volt \text{V}\, = \text{W}\,/\text{A}\, = \text{kg}\,\text{m}\,^{2}/\text{A}\,\text{s}\,^{3}
Charge Q:= \int I \,\text{d}t Coulomb \text{C}\, = \text{A}\,\text{s}\,
Resistance R := \frac{\,\text{d}U}{\,\text{d}I} Ohm \Omega\, = \text{V}\,/\text{A}\, = \text{kg}\,\text{m}\,^{2}/\text{A}\,^{2}\text{s}\,^{3}
Capacitance C:= \frac{\,\text{d}Q}{\,\text{d}U} Farad \text{F}\, = \text{C}\,/\text{V}\, = \text{A}\,^{2}\text{s}\,^{4}/\text{kg}\,\text{m}\,^{2}
Inductance L := \frac{\,\text{d}\Phi}{\,\text{d}I} Henry \text{H}\, = \text{V}\,\text{s}\,/\text{A}\, = \text{kg}\,\text{m}\,^{2}/\text{A}\,^{2}\text{s}\,^{2}
{}^{\text{Magnetic}}_{\textstyle \text{Flux}} \Phi_{\mathrm M} := \int \boldsymbol B \,\text{d}\boldsymbol A Weber \text{Wb}\, = \text{V}\,\text{s}\, = \text{kg}\,\text{m}\,^{2}/\text{A}\,\text{s}\,^{2}
{}^{\text{Magnetic}}_{\textstyle \text{Flux Density}} \boldsymbol B := \mu_0 \boldsymbol H Tesla \text{T}\, = \text{Wb}\,/\text{m}\,^{2} = \text{kg}\,/\text{A}\,\text{s}\,^{2}
